# 环境配置

本技能所有的工具调用都由 AI 自动执行，你只需要提前配好以下环境变量。

## 字节豆包 Seedream AI 配图（火山方舟 Ark API）⭐ 推荐

```bash
# 将 API Key 填入 skill 目录下的 .env 文件中：
# C:\Users\<你的用户名>\.codex\skills\material-search-assistant\.env
# 格式：
ARK_API_KEY=你的API密钥
```

默认模型：`doubao-seedream-5-0-260128`
默认尺寸：`2K`
脚本入口：`scripts/render_ark_images.py`
获取密钥：https://console.volcengine.com/ark/

## 通义万相 AI 配图（DashScope API）备选

```bash
# macOS / Linux
export DASHSCOPE_API_KEY="你的API密钥"

# Windows PowerShell
$env:DASHSCOPE_API_KEY="你的API密钥"
```

获取密钥：https://dashscope.aliyun.com/

不需要装 Python 库——技能脚本自带依赖处理。

## 其他工具

以下工具由技能内置调用，你不需要手动安装或配置：
- **universal-transcriber** — 视频下载（B站/抖音/小红书/YouTube）
- **Playwright** — 网页截图
- **Remotion** — 动画生成
- **yt-dlp** — 视频下载后备方案

**这些 AI 会自动处理，不需要你动手。**
