# AI 视频生成 Prompt 写作规范

写视频 Prompt 时，不要只描述画面"是什么"，还要描述画面"怎么动""怎么变化""最后停在哪里"。对于需要准确的文字、数字、时间、图标，不要交给 AI 视频生成，要让视频模型只生成抽象背景和视觉氛围，文字与信息动画后期用代码或剪辑完成。

---

## 一、视频 Prompt 和图片 Prompt 最大的区别

图片 Prompt 只需要描述"一帧画面长什么样"。

视频 Prompt 必须描述：

- 画面是什么
- 画面怎么动
- 镜头怎么动
- 元素如何变化
- 情绪如何递进
- 哪些东西必须不要出现
- 后期要不要留白
- 是否允许文字、数字、人脸、真实场景

## 二、标准结构

一个高质量视频 Prompt 按以下顺序写：

```
1. 画面类型 + 风格
2. 主题含义
3. 主体元素
4. 构图位置
5. 动作变化
6. 镜头运动
7. 色彩和质感
8. 情绪氛围
9. 后期需求
10. 禁止内容
11. 技术参数
```

## 三、通用公式

```
[视频类型] + [视觉风格] + [主题表达]。

画面中有 [主体元素A]、[主体元素B] 和 [主体元素C]，
它们位于 [构图位置]，
以 [运动方式] 发生变化，
从 [起始状态] 逐渐变成 [结束状态]。

镜头采用 [镜头运动]，
整体色彩为 [色彩体系]，
质感为 [材质/纹理]，
氛围是 [情绪关键词]。

画面需要 [留白/无文字/可后期叠加/透明感等后期要求]。

不要出现 [负面内容]。

[比例]，[清晰度]，[时长]，[运动强度]。
```

## 四、各部分写法细则

### 4.1 画面类型

开头要明确告诉模型你要的是视频，不是照片。

推荐：`黑白抽象平面设计视频`、`现代公益纪录片科技视觉`、`非写实插画风格视频`、`极简信息图动画风格视频`

不要只写：`DNA`、`孩子`、`地图`、`数据库`

### 4.2 主题表达

告诉模型这段画面表达什么含义。

例：`表现"DNA自动比对并帮助确认身份"的过程。`、`表现"千千万万条回家的路"的概念。`

主题表达要简洁，但必须有。否则模型只会机械拼元素。

### 4.3 主体元素

主体元素要写成"符号"和"结构"，而不是现实物。

❌ `真实的实验室里，医生拿着血样进行DNA检测。`
✅ `两组抽象DNA线条和白色数据点，缓慢进入中央数据库核心。`

❌ `一个孩子被人带走。`
✅ `一个小小的白色儿童背影剪影，缓慢向远处移动，逐渐变小、变淡。`

### 4.4 构图位置

主动要求留白，方便后期叠加信息。

常用构图写法：
- `画面中央保留大面积干净留白，方便后期叠加巨大倒计时数字。`
- `画面右侧保留留白，方便后期叠加说明文字。`
- `主体位于画面左侧，右侧是抽象地图网格。`
- `中心是数据库核心，左侧是DNA信息，右侧是地图定位区域。`

### 4.5 动作变化

必须写清楚：什么东西在动、往哪里动、速度快慢、从什么状态变成什么状态、最后停在哪里。

例：`两组DNA线条从画面左侧缓慢向中央数据库核心移动。`、`进入数据库后，点阵开始有节奏地排列和比对。`、`匹配成功时，中心出现柔和白色圆环扩散。`

### 4.6 镜头运动

公益纪录片、平面设计、抽象风，镜头运动不宜太猛。

推荐：`镜头缓慢推进。`、`像二维海报被轻轻推近。`、`轻微二维视差运动。`、`整体运动低频、平稳、克制。`

不推荐：`快速旋转镜头`、`剧烈摇晃`、`高速穿梭`、`爆炸式转场`、`震撼大片感`

### 4.7 色彩和质感

写成完整体系，不要只写"黑白"。

`黑白高对比，少量柔和灰阶，纸张纹理，轻微胶片颗粒，平面插画感。`
`整体只有黑、白、灰三种层次，强负空间，细线条，几何块面，丝网印刷质感。`
`整体明亮、干净、理性、有希望，不要阴暗恐怖。`

### 4.8 情绪词

不要用"高级"这种虚词。用：`克制`、`安静`、`理性`、`温和`、`有希望`、`庄重`、`清晰`、`沉默`、`有秩序`、`不煽情`、`不惊悚`

### 4.9 后期需求

素材可能要进 Remotion 继续做动效，必须写：
- `不要生成文字。`
- `不要生成数字。`
- `画面保留留白，方便后期叠加中文文字。`
- `适合后期叠加白色路径线、图标和倒计时。`
- `不要字幕，不要水印，不要logo。`

### 4.10 禁止内容（负面 Prompt）

按项目风险分类排除：

- 跑成写实：`photorealistic, realistic street, live action, real people`
- 跑成恐怖：`horror, thriller, creepy atmosphere, dark horror`
- 真实儿童：`real child face, crying child, identifiable person`
- 错误文字：`text, subtitles, watermark, logo`
- 赛博/医疗：`cyberpunk, neon blue, realistic laboratory, doctor, blood sample, syringe`

## 五、负面 Prompt 写法规范

两段式结构：通用排除 + 段落专属排除。

```
通用：photorealistic, realistic street, live action, real people, real child face, identifiable person, crying child, kidnapping scene, criminal, violence, blood, horror, thriller, creepy atmosphere, dark horror, medical thriller, realistic laboratory, doctor, syringe, blood sample, police chase, weapon, red warning light, cyberpunk, neon blue, code rain, 3D render, glossy texture, colorful image, text, subtitles, watermark, logo, low quality, blurry, distorted objects, messy composition

DNA段落加：realistic laboratory, doctor, blood sample, syringe, microscope close-up, hospital horror, medical thriller

孩子段落加：real child face, crying child, adult grabbing child, kidnapping scene, violence

法律段落加：real courtroom, judge face, prison, crime scene, blood, real legal document text
```

## 六、Prompt 长度建议

| 项目 | 建议范围 |
|------|---------|
| 中文视频 Prompt | 250-500字 |
| 英文视频 Prompt | 180-350词 |
| 负面 Prompt | 50-120个词 |

太短模型不知道怎么动，太长模型抓不住重点。

---

## 七、成品 Prompt 检查清单

- [ ] 是否明确写了"视频"？
- [ ] 是否明确写了"非写实"？
- [ ] 是否明确写了色调风格和质感？
- [ ] 是否写了主体在画面中的位置？
- [ ] 是否写了元素如何运动？
- [ ] 是否写了镜头如何运动？
- [ ] 是否写了开始状态和结束状态？
- [ ] 是否要求留白，方便后期叠加信息？
- [ ] 是否禁止AI生成文字、字幕、数字？
- [ ] 是否排除了不适宜内容（写实、恐怖、血腥等）？
- [ ] 是否写了比例、时长、清晰度？
- [ ] 是否提供了负面 Prompt？

## 八、输出模板

写 AI 视频 Prompt 时按以下结构输出到 `.md` 文件：

```markdown
# [分镜编号] [主题描述]

## 画面类型
[黑白抽象平面设计视频 / 非写实插画风格视频 / ...]

## 推荐工具
可灵 / 即梦 / Sora

## 时长建议
[5-8秒]

---

## 中文视频 Prompt

[250-500字，按通用公式写：画面类型+视觉风格+主题表达→主体元素+构图+动作变化→镜头运动+色彩质感+情绪→后期需求→禁止内容→技术参数]

---

## 英文视频 Prompt

[180-350词，与中文Prompt对应]

---

## 负面 Prompt

[50-120个词]

---

## 建议参数

- 比例：16:9
- 时长：[X]秒
- 运动强度：低频 / 平稳 / 克制
- 画质：超高清

---

## 写作思路

[简述这段prompt的结构和为什么这样写]
```
