# 输出规范

所有素材和中间文件，全部放在 `material-pack/` 这一个文件夹内。

## 文件夹结构

```
material-pack/
  ├── 视频/              ← 下载的视频文件 (.mp4)
  ├── 图片/              ← 截图 + 搜索下载的图片 (.png/.jpg)
  ├── 动画/              ← Remotion 渲染的动画 (.mp4)
  ├── ai配图/            ← 通义万相生成的 AI 配图 (.png)
  ├── ai视频/            ← AI视频 prompt 文件 (.md)
  ├── prompts/           ← 临时 prompt 文件（生成完马上删）
  └── 分镜清单.md        ← 文案 vs 素材对照表
```

**铁律**：`material-pack/` 外不留任何文件。临时文件放里面，用完就清。

## 不要的文件夹

这些一律不建：
- 不要 `99_备选未用/` — 不合格的直接删
- 不要按分镜再分文件夹（`01_xxx/`、`02_xxx/`）
- 不要 `素材日志.csv`、`视觉决策.md` 之类中间文档

## 文件命名

```
分镜编号_内容描述_来源.后缀
```

示例：
- `02_DynaTAC产品正视图_Google图片.jpg`
- `03_Nokia3310经典造型_百度百科.png`
- `02_Cooper首次通话_历史图库.png`
- `01_科技时间线动画.mp4`

规则：
1. 不用英文命名
2. 内容描述写具体，不只是"相关图片"
3. 来源写平台名简写（B站/YouTube/Google/wiki/微博...）
